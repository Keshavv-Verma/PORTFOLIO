## 🚀 How to Run

1. Navigate to the project folder: `c:\Users\Asus\Keshav-Portfolio`
2. Double-click `index.html` to open in your default browser

### Option 2: Live Server (Recommended)
1. Install Live Server extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

```bash

python -m http.server 8000
# Then open http://localhost:8000 in your browser
```

### Option 4: Node.js Server
```bash

# Follow the URL shown in terminal
```

## 📝 Customization Guide

### Change Colors
Edit CSS variables in `index.css`:
```css
:root {
    --accent-primary: #6366f1;  /* Change to your color */
    --accent-secondary: #8b5cf6;
    --accent-tertiary: #ec4899;
}
```

### Update Typing Animation
Edit the phrases array in `index.js`:
```javascript
const phrases = [
    'Your Title 1',
    'Your Title 2',
    'Your Title 3'
];
```

### Add More Projects
Copy a project card in `index.html` and update the content.
